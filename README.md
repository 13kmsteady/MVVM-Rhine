# MvvmArchitecture(building...)

MVVM模式的开发框架。

### 开发环境：

AndroidStudio 3.0.0

### 网络层：

HTTP : Rxjava2 + Retrofit2 + Okhttp3

ImageLoader:Glide v4

Cache: RxCache

### 数据处理层：

JavaBean: lombok

Java8：Stream + Lambda

Dependencies Injection: DaggerAndroid

### UI层：

Views Binding: DataBinding

List: RecyclerView + MultiType

Layout:ConstraintLayout
